Welcome to Ringtools Web. Although there is quite some nice functionality, it is not without issues. Please report them on GitHub if you encounter them, enjoy!

How does this work

Ring Participants and their information (loaded from the CSV) will be stored in your browser (LocalStorage)
Node information based on the public key of the nodes will be retrieved from LND node without any channels, this might result in slower updates because of less gossip.

Known issues

  • If you directly load the ringtools on the design/watch page the node-information is not loaded yet, switch to the home/settings page and back to make it work.
  • Channel updates broke after implementing ngrx state.
  • The custom LND PubSub server setting is not working yet, since the library is not supporting easy switching.
